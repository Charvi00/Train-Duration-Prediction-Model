# Train-Duration-Prediction-Model

A Machine Learning based web application to predict train travel time using distance and stops.

## Features
- Random Forest Model
- Flask Web App
- Real-time Prediction
- Responsive UI

## Tech Stack
- Python
- Flask
- Scikit-Learn
- HTML/CSS
